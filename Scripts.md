# Scenario Building Scripts

These are some scripts that you can run to help make your scenario.  These files are run using the `Load Script` button in the Lua Console, and do not need to be included in the final product.

## Polygon Maker

The polygon maker script helps you make polygons to use with the General Library function `gen.inPolygon`.  Load `PolygonScript.lua` from within the `Scripts` directory.

Save your game before running this script.  It will place units on the map in an effort to show you what is and is not covered by your polygons.
