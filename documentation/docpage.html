Creates a unitType unit or units for tribe at one or more of the locations, with details determined by the options table.

Arguments

  1. : The type of unit to be created.
  2. : The tribe owning unit(s) being created.
  3. : The tile(s) where the unit(s) will be created.
  4. : The tile(s) where the unit(s) will be created.
    • "count"=:The number of units to create. Nil means create one unit.
    • "randomize"=:If true, randomize the list of locations. If false or nil, try to place the unit(s) on the tile with the smallest index first.
    • "scatter"=:If true, and if randomize is true, each unit is created on a random tile in the location table.
