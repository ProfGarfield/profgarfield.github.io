The ‘nil’ data type has a single value nil, and tends to represent the absence of a value. Nil is a data type which is native to Lua, and is considered ‘falsy’ in logical statements. Unassigned keys in a table return nil as their value. Programming in Lua, 2.1 – Nil Tutorialspoint Lua – Data Types
